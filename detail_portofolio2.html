App Kawan Ternak

Aplikasi ini merupaan aplikasi yang dapat membantu peternak untuk merecording hewan ternaknya, dengan recording yang baik maka kualitas ternak juga akan terus meningkat, aplikasi ini juga akan memberikan rasa aman kepada peternak dikarenakan semua data yang diinputkan oleh peternak akan tersimpan di database.

Link Code

Deskripsi

Aplikasi ini merupakan project aplikasi yang cukup kompleks dimana aplikasi ini memiliki beberapa fitur seperti fitur recording data hewan ternak, keuangan, edukasi, artikel, dan forum diskusi. Aplikasi ini menggunakan teknologi QR Code untuk melakukan identifikasi hewan ternak hal tersebut berguna untuk meng efisienkan waktu yang digunakan dalam melakukan recording hewan ternak, jadi cara kerja dari fitur utama aplikasi ini (recording data hewan ternak) yaitu ketika user menambahkan data hewan ternak maka data akan masuk kedalam database sekaligus akan menggenerate sebuah QR code dari data tersebut, lalu ketika user ingin mendapatkan QR code tersebut maka user harus melakukan request dengan menghubungi admin , setelah itu admin akan mengirimkan QR code tersebut melalui notifikasi user tersebut.
Adapun fitur yang terdapat pada aplikasi ini adalah sebagai berikut :

1. Identifikasi hewan ternak dengan cara scan QR Code

2. Artikel

3. Forum Diskusi

4. Edukasi

5. Manajemen Keuangan

6. Form Recording Hewan Ternak

7. Notification

8. Edit Profil
