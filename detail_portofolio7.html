App Loved Things

Deskripsi

Aplikasi Loved Things merupakan aplikasi yang menyediakan tempat jual beli barang-barang thrifting, untuk melakukan jual beli tersebut penjual ataupun pembeli harus daftar terlebih dahulu untuk mengakses aplikasi ini, untuk payment activity aplikasi ini masih bersifat demo dengan kata lain tidak menggunakan uang sungguhan. Aplikasi ini dibuat dengan metode MVP.

Aplikasi ini di rancang dan dibangun sekitar 1-2 minggu.

Fitur yang terdapat pada aplikasi ini :

1. Search Barang

2. CRUD Barang

3. Personal Chat

4. Keranjang

5. Edit Profil

6. Filter Kategori Barang

Tools yang digunakan :

1. Android studio

2. Firebase console

Database :

1. Firebase (Firebase Realtime Database, Firebase Auth, Firebase Storage)

Bahasa pemrograman yang digunakan

1. Java

Metode pengembangan Extreme Programming

1. Planning

2. Design

3. Coding

4. Testing

5. Release
