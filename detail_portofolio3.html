App Aset SMKN Situraja

Aplikasi ini merupaan aplikasi monitoring dan memanage peminjaman aset SMKN Situraja, demana dengan adanya aplikasi ini user atau siswa dapat terbantu dalam melakukan peminjaman aset dan pengelola aset pun dapat melakukan monitring terhadap aset yang dipinjam oleh para siswa.

Deskripsi

Aplikasi ini merupakan project aplikasi yang cukup kompleks dimana pada aplikasi ini terdapat 2 type user yaitu user (siswa) dan admin (pengelola), Aplikasi ini dirancang untuk membantu para siswa dalam melakukan peminjaman aset dan memudahkan pengelola aset dalam memonitoring aset yang ada di SMKN Situraja. Untuk cara kerjanya admin harus memasukan data barang kedalam database lewat aplikasi ini, lalu user dapat melihat daftar barang yang telah dimasukan, setelah itu user dapat melakukan peminjaman melalui aplikasi ini dan admin akan mengecek dan mempersiapkan terlebih dahulu barang yang akan dipinjam tersebut lalu setelah barang siap admin akan acc permintaan tersebut dan user akan mengambil barang tersebut kepada pengelola, setelah admin melakukan acc maka waktu peminjaman akan dimulai dengan hitungan mundur dari tanggal peminjaman sampai tanggal perjanjian pengembalian, lalu ketika user melakukan pengembalian barang user harus foto barang tersebut dan kirimkan pengajuan pengembalian di aplikasi ini. Dengan begitu user tidak harus antri dan cari barang yang akan di pinjam, dan admin akan lebih mudah untuk memonitoring aset yang ada di SMKN Situraja. Dalam pengembangan aplikasi ini saya berperan aktif sebagai fullstack developer, dimana saya mengerjakan project ini dari mulai membuat sebuah flow-wireframe-mockup-front end-back end.
