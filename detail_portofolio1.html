App Suketdes

Aplikasi Suktdes merupakan aplikasi pelayanan surat keterangan desa berbasis android yang saya rancang untuk salah satu tugas dalam salah satu mata kuliah, dimana aplikasi ini dapat digunakan kapanpun dan dimanapun karena aplikasi ini tidak memerlukan sinyal internet (offline). Dengan begitu masyarakat desa khususnya tidak perlu datang ke kantor desa.

Deskripsi

Project aplikasi ini dirancang oleh saya sendiri dalam rangka memenuhi tugas salah satu matakuliah dan aplikasi ini masih dalam pengembangan (belum di publikasikan secara luas). Dalam perancangannya aplikasi Suketdes ini dirancang menggunakan Android Studio dengan bahasa pemrograman Java. Cara kerja dari aplikasi suketdes ini sangat mudah yaitu user hanya perlu memilih salah surat keterangan apa yang ingin dibuat, setelah itu user tinggal mengisi semua form yang telah disediakan, dan terakhir klik tombol buat maka surat keterangan tersebut akan otomatis dibuat dalam format PDF di directory Download user. Aplikasi ini dirancang menggunakan metode Agile dimana tahapan yang dilakukan seperti plainning, design, coding, testing, review.
Adapun fitur dari aplikasi ini adalah sebagai berikut :

1. Membuat Surat Keterangan Desa
